# Giới thiệu dự án
Dự án này sử dụng vi điều khiển **Arduino Nano** để đọc dữ liệu từ các cảm biến nhiệt độ tương tự **LM35** một cách đồng thời và chính xác.

### 🛠 Các kỹ thuật nổi bật:
* **Phần cứng:** Thiết kế và giao tiếp mạch cảm biến LM35 với các chân ADC.
* **Xử lý dữ liệu:** Định dạng và đóng gói dữ liệu cảm biến thông minh bằng hàm `sprintf()` trong C/C++.
* **Quản lý mã nguồn:** Áp dụng hệ thống quản lý phiên bản Git để theo dõi thay đổi phần mềm (sử dụng thành thạo `git commit`, `git push`, và tra cứu lịch sử code với `git blame`).
